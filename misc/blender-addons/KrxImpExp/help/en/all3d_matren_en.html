Material Autorenaming

To allow Spacer to work with materials correctly the materials must have names which are known to Spacer. You can set your materials to correct names by yourself (in 3D Editor) or you can use an autorenaming tool. I mean the 3ds file exporter will set materials' names according to diffuse texture map. For example, if a material uses diffuse map "ow_icedragon_lake_01.tga" then it probably must have name "lake_env_map_and_lookthrough_01". The autorenaming will give the material this name. To activate autorenaming, check the "Rename materials" checkbox, as shown at the picture below.

Probably you should also click the "Parameters" button to setup the autorenaming tool.

Click button "..." and specify path to the "matlib.ini" file. This file contains information about pml files which contain description of all materials known to Spacer. Then click button "Load materialfilter". And click button "OK" to apply changes.

Note 1. The autorenaming doesn't change names of materials which are without diffuse texture map. The autorenaming doesn't change materials' names beginning with "P:".

Note 2. The autorenaming sets names of materials which are exported to 3ds file. It doesn't change names of materials in the current 3D scene.
